

This is an implementation of the game, Tetris.

Run tetris.py in terminal to play.

No 3rd party packages are required to run.

This only works on MS Windows due to the way certain
features are implemented.


Game Controls

    UP, X   : rotate clockwise
    Z       : rotate counter clockwise
    SPACE   : hard drop
    DOWN    : soft drop
    LEFT    : move left
    RIGHT   : move right
    ESC     : quit
